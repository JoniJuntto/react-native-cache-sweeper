# react-native-cache-sweeper

Cache sweeper cleans the app cache

## Installation

```sh
npm install react-native-cache-sweeper
```

## Usage

```js
import { multiply } from 'react-native-cache-sweeper';

// ...

const result = await multiply(3, 7);
```
